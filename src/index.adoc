// introdução

A principal meta deste manual é fornecer uma extensa introdução ao  http://www.starling-framework.org[Starling Framework].
Starling é um motor de desenvolvimento multiplataforma para  _ActionScript 3_ que pode ser usado por todos os tipos de aplicações, com um foco especial para jogos 2D.

ifdef::backend-html5[]
TIP: Fique à vontade para baixar a link:manual.pdf[versão em PDF] deste documento 
endif::[]

Neste guia, você vai aprender:

* As tecnologias por trás do Starling e os princípios que ele segue.
* Como escolher uma IDE e configurar seu primeiro projeto.
* Os conceitos básicos como a lista de exibição, eventos e o sistema de animação.
* Técnicas avançadas, como explorar o potencial de programas de fragmento e programas de vértice.
* Como extrair a melhor performance deste framework.
* O que é necessário para rodar seu jogo em aparelhos celulares e tablets.

Continue lendo para mergulhar de cabeça neste framework!
